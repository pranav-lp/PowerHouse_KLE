
  # 🚀 AI for Longevity web app


An AI-powered health insights platform designed to evaluate lifestyle habits and provide personalized recommendations to improve long-term health and longevity.

📌 Overview

This project is a modern web application that collects user health data through an interactive questionnaire and generates:

🫀 Heart Health Score
🤖 AI-driven insights
📊 Visual health analytics
💬 Chat-based health assistant

Built with a focus on user experience, modular UI components, and scalable frontend architecture.

🧠 Key Features
Health Questionnaire Flow
Multi-step form with progress tracking
Structured health & lifestyle data collection
AI Insights Engine
Generates personalized recommendations
Interprets user inputs for actionable feedback
Heart Health Score
Evaluates user inputs to produce a score
Designed for quick health risk awareness (Note: Not medical advice)
Interactive Dashboard
Clean UI with charts and visual summaries
Real-time feedback
AI Chatbot
Assists users with health-related queries
Enhances engagement and usability

⚙️ Installation & Setup
1. Clone the repository
git clone <your-repo-url>
cd <project-folder>
2. Install dependencies
npm install
3. Run development server
npm run dev
4. Open in browser
http://localhost:5173
⚠️ Important Notes
This app provides AI-generated health insights, which may not always be medically accurate.
The heart health score is heuristic-based, not clinically validated.
Should be treated as a wellness tool, not a diagnostic system.
